# Markdown

## 基本语法
* 标题
    > #### 标题
* 无序列表
    > * 第一项
    > * 第二项
* 有序列表
    > 1. 第一项
    > 2. 第二项
* 区块引用
    > 解释说明
* 分割线
    > ---
* 链接
    > [百度](https://www.baidu.com/)
* 图片
    > ![图片](https://www.baidu.com/img/bd_logo1.png)
* 代码框
    > * 单行代码 `printf("Hello World!");`
    > * 多行代码
    ```
    for(int i = 0; i < 2; i++)
    {
        printf("Hello World!");
    }
    ```
* 表格
| ID | 姓名 | 年龄 | 
| :- | :-: | -: |
| 1 | Miaokit | 29 | 
| 2 | Utopiatek | 2 |
* 强调
    > _倾斜_，__加粗__
* 转义
    > \\ \' \~ \* \_ \- \+ \. \!
* 删除线
    > ~~删除~~
